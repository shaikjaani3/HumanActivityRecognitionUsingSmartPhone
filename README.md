# HumanActivityRecognitionUsingSmartPhone
Human Activity Recognition (HAR) using CNN : A deep learning project classifying actions (e.g., WalkingWithDog, TaiChi, Swing) from UCF50 videos. Built with TensorFlow/Keras, OpenCV; features frame extraction, CNN training, real-time prediction. Internship at Blackbuck Engineers, surpassing manual/sensor systems.
